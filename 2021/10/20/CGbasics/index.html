数值微分法（DDA）

已知P1(x1,y1)P2(x2,y2)P_1(x_1,y_1) P_2(x_2,y_2)P1​(x1​,y1​)P2​(x2​,y2​)，有：

  • k=(y2−y1)x2−x1k=\frac{(y_2-y_1)}{x_2-x_1}k=x2​−x1​(y2​−y1​)​

当∣k∣≤1|k|\leq 1∣k∣≤1：

  • x++; y += k;
    取像素点 (x, int(y + 0.5))

当∣k∣>1|k|>1∣k∣>1：

  • y++; x += 1 / k;
    取像素点 (int(x + 0.5), y)

中点画线法

假定直线斜率 0<k<10<k<10<k<1，并且直线上当前已确定一个像素点为 P(xp,yp)P(x_p, y_p)P(xp​,yp​)，则下一个与理想直线最接近的像素点只能是 P 点正右方的点 P1P_1P1​ 或右上方的点 P2P_2P2​ 两者之一。设 M 为线段 P1P2P_1P_2P1​P2​ 的中点，Q 为理想直线与线段 P1P2P_1P_2P1​P2​ 的交点。现需要确定下一个像素点。

  • 若M在Q上方，则P1离直线更近，应取P1为下一个像素。
  • 若M在Q下方，则P2离直线更近，应取P2为下一个像素

假设直线段起点为 (x1,y1)(x_1,y_1)(x1​,y1​)，终点为 (x2,y2)(x_2,y_2)(x2​,y2​)。
设直线方程：F(x,y)=ax+by+c=0F(x,y)=ax+by+c=0F(x,y)=ax+by+c=0
其中：

  • a=y1−y2a=y_1-y_2a=y1​−y2​
  • b=x2−x1b=x_2-x_1b=x2​−x1​
  • c=x1y2−x2t1c=x_1y_2-x_2t_1c=x1​y2​−x2​t1​

当 F(x,y)>0F(x,y)>0F(x,y)>0 时，则点在直线上方。
当 F(x,y)<0F(x,y)<0F(x,y)<0 时，则点在直线下方。
要判断 M 在 Q 的上方还是下方，只需把 M 代入 F(x, y)，并判断它的符号。

构造判别式：

d 的增量

d 是 xpx_pxp​ 和 ypy_pyp​ 的线性函数，可采用增量计算，以提高运算效率。
当 d≥0d\geq 0d≥0 时，取正右方像素 P1P_1P1​，则再下一个像素的判别式为：

    =a(xp+1)+b(yp+0.5)+c+a=d+a=a(x_p+1)+b(y_p+0.5)+c+a=d+a=a(xp​+1)+b(yp​+0.5)+c+a=d+a

此时 d 的增量为 a。
同理 d<0d<0d<0 时 d 的增量为 a+ba+ba+b。

d 的初值
  • d0=F(x1+1,y1+0.5)=a(x1+1)+b(y1+0.5)+cd_0=F(x_1+1, y_1+0.5)=a(x_1+1)+b(y_1+0.5)+cd0​=F(x1​+1,y1​+0.5)=a(x1​+1)+b(y1​+0.5)+c
    =(ax1+by1+c)+a+0.5b=F(x1,y1)+a+0.5b=(ax_1+by_1+c)+a+0.5b=F(x_1, y_1)+a+0.5b=(ax1​+by1​+c)+a+0.5b=F(x1​,y1​)+a+0.5b

伪代码
  1. 输入直线段的两个端点 P1(x1,y1)P_1(x_1, y_1)P1​(x1​,y1​) 和 P2(x2,y2)P_2(x_2, y_2)P2​(x2​,y2​)。
  2. 初始化：a = y1 - y2，b = x2 - x1，d = 2a + b，x = x1，y = y1，画点 (x, y)。
  3. 若 x < x2，则执行下列各步，否则算法结束。
  4. 判断 d 的符号：若 d < 0，则 (x, y) 更新为 (x + 1, y + 1)，d 更新为 d + 2a + 2b；
    否则 (x, y) 更新为 (x + 1, y)，d 更新为 d + 2a。
  5. 画点 (x, y)，返回 3。

Bresenham

核心思想依旧是构造判别式，令两个候选像素的 y 值与线段上理想 y 值的差值分别为 d1 和 d2，判断 d1 - d2 的符号。

伪代码
  1. 输入直线段的两个端点 P1(x1,y1)P_1(x_1, y_1)P1​(x1​,y1​) 和 P2(x2,y2)P_2(x_2, y_2)P2​(x2​,y2​)。
  2. 初始化：dx = x2 - x1, dy = y2 - y1, d0 = 2dy - dx，画点 (x, y)。
  3. 若 x < x2，则执行下列各步，否则算法结束。
  4. 判断 dk 的符号：若 dk > 0，则 (x, y) 更新为 (x+1, y+1)，dk 更新为 dk+2dy-2dx；
    否则 (x, y) 更新为 (x + 1, y)，d 更新为 dk+2dy。
  5. 画点 (x, y)，返回 3。
