音效

  • 首先需要一个挂载 Manager 的 Object
  • 上一个场景的 Manager 实例需要带到下一个场景中去，故使用 DontDestroyOnLoad(gameObject)
  • 调试时若不从第一个场景进入则不存在该 Object，十分不便
  • 若每个场景都放置该 bject，大量不会销毁的 Object 会带着其上挂载的的 Manager 堆积起来
  • 故在 Awake 中做判断销毁任何新加载的 Object
